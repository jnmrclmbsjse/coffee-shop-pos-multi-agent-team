# UCM Coffee Studio Brand Binding

The interface uses a cool, restrained light system with operational density, system UI typography, and a single green accent for action and selection.

## Core Six

```css
--bg: oklch(98% 0.005 250);
--surface: oklch(100% 0 0);
--fg: oklch(22% 0.02 240);
--muted: oklch(50% 0.018 240);
--border: oklch(90% 0.008 240);
--accent: oklch(58% 0.16 145);
```

## Type

- Display: `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- Body: `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- Mono: `ui-monospace, SFMono-Regular, Consolas, monospace`

## Observed Posture

1. Restrained light surfaces support bright working environments.
2. Green indicates action, selection, focus, or affirmative state, not decoration.
3. Thin cool borders organize dense data; working surfaces use 10px radii.
4. Controls are conventional, labeled, and at least 44px tall.
5. Errors and blocked actions explain what stayed unchanged and what to do next.
